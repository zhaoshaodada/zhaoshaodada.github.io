总结

vue-i18n 在 Vue 项目中的三种使用场景：

  1. 模板文本：{{$t('key')}}
  2. 组件方法属性：$t('key')
  3. JS 方法中：this.$i18n.t('key')

注意在异步回调中使用时需要提前保存 this 引用。
